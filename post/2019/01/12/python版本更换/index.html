树莓派Python版本更换

2019-01-12
嵌入式 树莓派 Python

树莓派自带Python2和Python3，其中默认是Python2，使用Python3的方法是Python3和Pip3等，由于Python2的种种原因，我计划干掉Python2，只用Python3.

  1. 删除Python2.7
1

sudo apt-get autoremove python2.7
  1. 链接python
1
2

sudo ln -s python3.5 python
sudo ln -s pip3 pip
  1. 检测
1

python
  1. 后期处理
1
2

sudo apt-get update
sudo apt-get upgrade
